Digital Booking

Digital Booking es una aplicación de práctica que permite crear, ver, editar y borrar Canales con el objeto de ver transmisiones directos o streams, utiliza React con componentes de clase, redux, portales, jsondb y rtmpserver para las transmisiones. Digital Booking es una extensa aplicación de práctica creada por un equipo de 6 integrantes, cuyo objetivo es concretar bookings de hospedajes o ofrecer sitios con el objetivo de rentarlos dependiendo del rol que tome el usuario, y muchas funcionalidades más. Utilizamos: -Gitlab, Jira, Scrum, discord y whatsapp para comunicarnos y organizar las actividades en grupo. -Terraform, gitlab y AWS para la infraestructura. -Java, postman y Mysql para el servidor BackEnd. -Con respecto a testing, test cases, exploratory tests, Cypress y Postman. -React, Materiaul UI, Zustand conforman las tecnologias para el front.
